## ğŸš€ Quick Start

1. Clone the repository:
   ```
   git clone https://github.com/nectariferous/Infinity-Web3-Crypto-Bot.git
   ```
2. Install dependencies:
   ```
   npm install
   pip install -r requirements.txt
   ```

   ```
   node server.js

   ```
